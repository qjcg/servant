# Servant

Simply serve HTTP.

**NOTE: This is pre-alpha software. Several
features are not yet fully implemented.**


## Features

- Serve HTTP my way
- That's it


## Usage

```shell
# Serve HTTP on localhost:8080
$ servant

# Serve ~/public via LAN IP and custom port:
$ servant -i 10.13.37.10 -p 8888 ~/public
```
